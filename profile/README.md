## Public service

The **delay replay UK** subscription service is available to the public: https://delayreplay.uk

Subscription requests can be submitted via the website for nightly delay reports on regular routes. All new subscriptions are reviewed before activation.

## This repository

This GitHub organisation contains the private source code and infrastructure used to operate **delay replay UK**.

Access is strictly by invitation only and is limited to trusted contributors.  
Requests to join are not accepted.

If you have received an invitation, you are in the correct place.

## About

**delay replay UK** is a small, independently run service focused on accuracy, privacy, and reliability.
It is not affiliated with any train operators or third parties.
